Arizona Well Pipe Design

Fluid Mechanics design problem for a well pumping system

Fig. 1 — Final design showing system curve and pump curve, with labeled pump head, flow rate, and operating point.

Fig. 2 — Output of filtered Python results using SCH 80 3-inch and 4-inch pipe options.

Fig. 3 — Symbolic solution for the system curve equation, including input values for schedule 80 4-inch pipe.

Objectives

  • Given a well in an Arizona pasture with defined depth, distance, available pump power, impeller diameters, and pipe size options, determine a viable pumping system design.
  • Select impeller diameter, pipe diameter, and RPM to satisfy all system constraints.

Outcomes & Contributions

  • Recommended design: schedule 80 3-inch pipe, 3.25-inch diameter impeller, at 3,539 RPM.
  • Computed operating points in Python by iterating over impeller diameter and RPM combinations, then filtering against design constraints.

Technical Details & Skills

  • Python scripting for parametric analysis and constraint-based filtering of operating points.
  • Applied fluid mechanics principles to analytically solve for system curve intersections and maximum pressure to identify valid operating points.
